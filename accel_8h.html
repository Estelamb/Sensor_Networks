Detailed Description

I2C interface for a 3-axis accelerometer.

This module provides initialization, raw data reading, and conversion functions for a 3-axis accelerometer over I2C. Supports setting measurement range and converting raw data to g or m/s².

Read raw accelerometer values for X, Y, and Z axes.

Each axis is 14-bit, left-aligned in MSB/LSB registers. This function shifts right by 2 bits to obtain 14-bit signed values.

Parameters
devPointer to I2C device descriptor.
xPointer to store X-axis raw value.
yPointer to store Y-axis raw value.
zPointer to store Z-axis raw value.
Returns
0 on success, negative errno code on failure.
