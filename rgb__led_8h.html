Detailed Description

Interface for controlling an RGB LED using GPIO pins.

This module provides functions to initialize and control an RGB LED connected via GPIO. Each color channel (Red, Green, Blue) is represented by a separate GPIO pin, allowing mixed color combinations through bitmask-based operations.

Write a bitmask value to the RGB LED pins.

Each bit controls one color channel:

  • Bit 0 → Red
  • Bit 1 → Green
  • Bit 2 → Blue
Parameters
rgb_ledPointer to the RGB LED descriptor structure.
valueBitmask (0–7) defining active color channels.
Return values
0If the operation succeeded.
-EIOIf a GPIO write failed.

Write a bitmask value to the RGB LED pins.

Each bit in the input value corresponds to a color channel:

  • Bit 0 → Red
  • Bit 1 → Green
  • Bit 2 → Blue

Example:

  • 0x1 → Red ON
  • 0x3 → Yellow (Red + Green)
  • 0x7 → White (all channels ON)

rgb_ledPointer to the RGB LED descriptor structure.
