Detailed Description

ADC interface layer for analog sensor sampling using Zephyr drivers.

This module provides a hardware abstraction layer for analog-to-digital conversion using Zephyr's ADC API. It defines configuration structures and utility functions for initializing ADC channels and retrieving raw, normalized, or voltage-converted readings.

Each sensor using the ADC should have its own configuration instance.
